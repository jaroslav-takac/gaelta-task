# Gaelta – Technical Assignment (E-commerce Developer)

This repository contains my solution for the technical assignment as part of the hiring process at Gaelta.

## Structure

- **task1-shoptet/**  
  Custom component adjustments for Shoptet product cards (HTML/CSS/JS snippet)

- **task2-api/**  
  API integration proposal (Shoptet webhook → Google Sheets)

- **task3-debug/**  
  Debugging and fixing JavaScript issues

- **task4-answers/**  
  Short theoretical answers to e-commerce related questions

- **task5-platforms/**  
  Platform-specific solutions (WordPress/WooCommerce, Shoptet, Upgates)

## Approach

The goal of this assignment was not only to deliver working code, but also to demonstrate:
- problem-solving approach
- understanding of e-commerce platforms
- ability to design scalable solutions

Each task contains both implementation and explanation where relevant.
